 fzc-animated-ship

Polymer element to display an animated F-Zero Climax ship

Install

bower install fzc-animated-ship

Usage

You should already know how to setup your page to use Polymer elements. If not, please consult the Polymer project documentation.

After setting everything up, just include the <fzc-animated-ship></fzc-animated-ship> tag, and you should see the following ship.

Options

By default a spinning Blue Falcon will be displayed, but you can specify which ship you want by passing the name to the ship attribute.

The current available ships are as follows:

  • Blue Falcon: <fzc-animated-ship ship="blue-falcon">
  • Golden Fox: <fzc-animated-ship ship="golden-fox">
